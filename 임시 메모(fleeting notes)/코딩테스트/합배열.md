합 배열
- 기존의 배열을 전처리한 배열
- 합 배열을 미리 구해놓으면 기존 배열의 일벙 범위의 합을 구하는 시간 복잡도가 O(N)에서 O(1)로 감소
- A[i]부터 A[j]까지 배열 합을 합 배열 없이 구하는 경우
	- 최악의 경우 i가 0이고, j가 N인 경우
	- 시간 복잡도가 O(N)
- 합 배열의 공식
```java
S[i] = S[i-1] + A[i]
```

구간 합
- i에서 j까지 구간 합을 구하는 공식
```java
S[j] - S[i - 1]
```
